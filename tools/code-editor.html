How to Run Your Code
First, open your system terminal (Terminal on Mac, CMD/PowerShell on Windows) and cd into your project folder or codespace before running any commands below. cd /path/to/your/project

HTML / CSS

Save your file and open it directly in any browser. No terminal needed. Or use Live Server in VS Code for auto-reload.

open index.html

JavaScript

Include in an HTML file with <script> tag, or run with Node.js:

cd your-project
node filename.js

Python

Open terminal, cd into your folder, then run:

cd your-project
python3 filename.py

React / Next.js

Open terminal, cd into your project folder, then:

cd my-react-app
npm install
npm run dev

Node.js / Express

Open terminal, cd into your project folder, then:

cd my-server
npm install
node server.js

C / C++

Open terminal, cd into your folder, compile and run:

cd your-project
gcc file.c -o out && ./out
g++ file.cpp -o out && ./out

C#

Open terminal, cd into your project, then use .NET CLI:

cd my-csharp-project
dotnet run

Swift

Open terminal, cd into your folder, then run:

cd your-project
swift filename.swift

Java

Open terminal, cd into your folder, compile and run:

cd your-project
javac Main.java && java Main

Rust

Open terminal, cd into your Cargo project, then:

cd my-rust-project
cargo run

Go

Open terminal, cd into your folder, then run:

cd your-project
go run main.go

Ruby

Open terminal, cd into your folder, then run:

cd your-project
ruby filename.rb

PHP

Open terminal, cd into your folder, then run or start server:

cd your-project
php filename.php
php -S localhost:8000

Docker

Open terminal, cd into folder with Dockerfile, then:

cd your-project
docker build -t app .
docker run app

Vue.js / Angular / Svelte

Start dev server:

npm install
npm run dev
