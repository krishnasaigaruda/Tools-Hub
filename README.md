# Tools Hub

Your all-in-one productivity toolkit — 60+ tools that run entirely in the browser. No accounts, no servers, no data collection. Everything stays on your device.

## Features

- **100% client-side** — works offline, no backend needed
- **No installation** — just open `Tools.html` in a browser
- **Privacy-first** — your files and data never leave your device
- **Dark theme** throughout
- **Saves work** via localStorage and IndexedDB

## Tech Stack

- Pure HTML, CSS, JavaScript — no build tools
- External libraries loaded from CDN:
  - [PDF.js](https://mozilla.github.io/pdf.js/) — PDF rendering
  - [pdf-lib](https://pdf-lib.js.org/) — PDF editing
  - [html2pdf.js](https://ekoopmans.github.io/html2pdf.js/) — HTML to PDF export
  - [Leaflet](https://leafletjs.com/) — Interactive maps
  - [Leaflet Routing Machine](https://www.liedman.net/leaflet-routing-machine/) — Route directions
  - [qrcode-generator](https://github.com/kazuhikoarase/qrcode-generator) — QR code generation
  - [Font Awesome](https://fontawesome.com/) — Icons
  - [Google Fonts](https://fonts.google.com/) — Typography

## Getting Started
